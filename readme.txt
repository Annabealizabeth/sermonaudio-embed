Easily embed SermonAudio.com sermons.

== Description ==
SermonAudio Embed uses WordPress' Embed API to allow you to easily embed sermons into your posts, pages, and custom post types simply by inserting a link to the sermonaudio.com sermon.

You can contribute to SermonAudio Embed here.  
[https://github.com/CarlosRios/sermonaudio-embed](https://github.com/CarlosRios/sermonaudio-embed)  

== Installation ==
Download SermonAudio Embed through your WordPress install's plugins directory.

1. Navigate to 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select `sermonaudio-embed.zip` from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard

= Using FTP =

1. Extract the `sermonaudio-embed` directory to your computer
2. Upload the `sermonaudio-embed` directory to the `/wp-content/plugins/` directory
3. Activate the plugin in the Plugin dashboard

== Frequently Asked Questions ==

= How do I use it? =
SermonAudio Embed embeds the SermonAudio.com media player in your posts by copying and pasting a link to the sermon like such: http://www.sermonaudio.com/sermoninfo.asp?SID=8413100417889

= Does it work with custom post types? =
Yes! SermonAudio Embed works anywhere that embeds are allowed in WordPress.
